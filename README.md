## <a name="introduction">Introduction</a>

Mediflow is a healthcare patient management application that allows patients to easily register, book, and manage their appointments with doctors, featuring administrative tools for scheduling, confirming, and canceling appointments, along with SMS notifications, all built using Next.js.

## <a name="tech-stack">Tech Stack Used</a>

- Next.js
- Appwrite
- Typescript
- TailwindCSS
- ShadCN
- Twilio
